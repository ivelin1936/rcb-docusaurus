# Renault Club Bulgaria — Onboarding Guide

Welcome to the **Renault Club Bulgaria (RCB)** platform. This guide covers every feature available in the system so that new members, administrators, and developers can get up to speed quickly.

## What Is RCB?

RCB is a full-stack web platform for the Renault Club Bulgaria community. It provides:

- Event registration and competition management
- Member garage and vehicle management
- Live chat and community polls
- News, gallery, and campaign publishing
- Reward points, ranks, badges, and leaderboards
- Admin tools for moderation, metrics, and configuration
