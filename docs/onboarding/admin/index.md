# Admin Guide

This section covers all administrative capabilities of the RCB platform. Admin functions require the `ROLE_ADMIN` role unless otherwise noted.

## Quick Navigation

| Area | Description |
|------|-------------|
| [User Management](./user-management) | List, search, lock, unlock, delete users; assign roles |
| [Member Lifecycle](./member-lifecycle) | Suspend, warn, and reinstate members |
| [Event Management](./event-management) | Create events, manage applications, results, disciplines |
| [Event Check-in (Admin)](./event-checkin-admin) | Scan QR check-in tickets at physical events |
| [News Management](./news-management) | Create and manage news articles |
| [Gallery Management](./gallery-management) | Manage photo albums and uploads |
| [Campaign Management](./campaign-management) | Create and manage club campaigns |
| [Partners Management](./partners-management) | Manage club sponsors and partners |
| [Chat Moderation](./chat-moderation) | Create chat rooms, delete messages, ban users |
| [Polls Management](./polls-management) | Create and manage community polls |
| [Membership Cards](./membership-cards) | Issue, update, and revoke membership cards |
| [Reward Configuration](./reward-config) | Configure point trigger values |
| [Feature Flags](./feature-flags) | Toggle platform features at runtime |
| [Dashboard & Metrics](./dashboard-metrics) | View KPI charts and export CSV reports |
| [Job Execution Logs](./job-execution-logs) | Monitor scheduled job execution history |

## Admin Access URL

The admin dashboard is accessible at `/admin` (requires `ROLE_ADMIN`). A **Quick Nav Tiles** panel provides shortcuts to all admin sections.
